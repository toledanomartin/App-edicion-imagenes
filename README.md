# App-edicion-imagenes

Una aplicación web simple para editar imágenes, permitiendo ajustar:
- Brillo
- Contraste
- Saturación
- Tono

## Características
- Carga de imágenes
- Edición en tiempo real
- Descarga de imágenes editadas
- Interfaz intuitiva y fácil de usar

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
